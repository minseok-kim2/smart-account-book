# 프로젝트 수행 보고서

## 💰 Spring Boot 가계부 프로젝트

---

## 1. 프로젝트 개요

| 항목 | 내용 |
|------|------|
| **프로젝트명** | 스마트 가계부 시스템 |
| **개발자** | 김민석 |
| **개발기간** | 2025년 12월 |
| **목적** | Spring Boot 기반 개인 재정 관리 웹 애플리케이션 개발 |

본 프로젝트는 Spring Boot 프레임워크를 활용하여 개인 가계부 시스템을 구현한 웹 애플리케이션입니다.
사용자가 수입/지출 내역을 등록, 조회, 수정, 삭제할 수 있으며, 월별 통계 및 카테고리별 지출 분석 기능을 제공합니다.
Chart.js를 활용한 데이터 시각화로 직관적인 소비 패턴 분석이 가능합니다.

---

## 2. 주요 기능

### 2.1 가계부 CRUD 기능

| 기능 | 설명 | URL |
|------|------|-----|
| **목록 조회** | 월별 수입/지출 내역 조회 | GET `/account` |
| **내역 등록** | 새로운 수입/지출 내역 등록 | GET/POST `/account/write` |
| **내역 수정** | 기존 내역 수정 | GET/POST `/account/edit/{id}` |
| **내역 삭제** | 내역 삭제 | GET `/account/delete/{id}` |
| **통계 조회** | 월별 통계 및 차트 | GET `/account/stats` |

### 2.2 핵심 차별화 기능

- **월별 요약**: 총 수입, 총 지출, 잔액을 한눈에 확인
- **카테고리별 분류**: 수입(급여, 용돈 등) / 지출(식비, 교통비 등) 카테고리 관리
- **데이터 시각화**: Chart.js를 활용한 도넛 차트 및 막대 차트
  - 카테고리별 지출 비율 파이 차트
  - 최근 6개월 수입/지출 추이 막대 차트
- **월별 필터링**: 원하는 월의 데이터만 조회 가능
- **반응형 UI**: Bootstrap 5를 활용한 모바일 대응 디자인

## 5. 데이터베이스 설계

### 5.1 ERD

```
┌─────────────────────────────────────┐
│              account                │
├─────────────────────────────────────┤
│ id            INT (PK, AI)          │
│ type          VARCHAR(10)           │ ← income/expense
│ category      VARCHAR(50)           │
│ amount        INT                   │
│ description   VARCHAR(200)          │
│ account_date  DATE                  │
│ created_date  TIMESTAMP             │
└─────────────────────────────────────┘
```

### 5.2 테이블 생성 SQL

```sql
CREATE TABLE account (
    id INT NOT NULL PRIMARY KEY AUTO_INCREMENT,
    type VARCHAR(10) NOT NULL COMMENT '수입/지출 구분 (income/expense)',
    category VARCHAR(50) NOT NULL COMMENT '카테고리',
    amount INT NOT NULL COMMENT '금액',
    description VARCHAR(200) COMMENT '설명',
    account_date DATE NOT NULL COMMENT '거래일자',
    created_date TIMESTAMP DEFAULT CURRENT_TIMESTAMP() COMMENT '등록일시'
);

-- 샘플 데이터
INSERT INTO account (type, category, amount, description, account_date) VALUES
('income', '급여', 3000000, '12월 급여', '2025-12-01'),

('expense', '통신비', 55000, '핸드폰 요금', '2025-12-07');
```

## 6. 실행 방법

### 6.1 필수 환경

- **JDK 21** 설치
- **MySQL 8.x** 설치 및 실행
- **STS4 (Spring Tool Suite 4)** 또는 IntelliJ IDEA

### 6.2 데이터베이스 설정

1. MySQL에서 데이터베이스 생성:
```sql
CREATE DATABASE jwbookdb;
USE jwbookdb;
```

2. account 테이블 생성 (위 SQL 실행)

3. `application.properties` 설정 확인:
```properties
spring.datasource.url=jdbc:mysql://localhost:3306/jwbookdb
spring.datasource.username=root
spring.datasource.password=qwer1234
```

### 6.3 실행

1. STS4에서 프로젝트 Import
2. `JwbookApplication.java` 실행
3. 브라우저에서 `http://localhost:8091` 접속

### 7.1 메인 페이지 (`/mainStart`)
- 프로젝트 소개 및 기술 스택 표시
- 가계부 바로가기 버튼

## 8. 기대효과

1. **실용적 기능**: 실제 사용 가능한 개인 재정 관리 도구
2. **데이터 시각화**: Chart.js를 활용한 직관적인 통계 표현

## 9. 기존 계획 대비 변경 사항

| 구분 | 기존 계획 | 최종 결과 | 변경 사유 |
|------|----------|----------|----------|
| 프로젝트명 | Spring Boot 게시판 시스템 | 스마트 가계부 시스템 | 다른 학생들과 차별화 |
| 주요기능 | 게시글 CRUD | 수입/지출 관리 + 통계 분석 | 실용성 및 기능 확장 |
| 추가기능 | 조회수 카운트 | Chart.js 데이터 시각화 | 포트폴리오 경쟁력 강화 |

**변경 사유**: 기본 게시판 프로젝트는 많은 학생들이 선택하는 주제이므로, 
차별화를 위해 가계부 시스템으로 변경하였습니다. 
Chart.js를 활용한 데이터 시각화 기능을 추가하여 포트폴리오로서의 가치를 높였습니다.
